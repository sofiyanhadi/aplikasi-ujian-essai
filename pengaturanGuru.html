Pusat Persiapan Ujian Guru

Siapkan ujian untuk kelas Anda dalam 3 langkah mudah.

Langkah 1: Siapkan Spreadsheet Soal

Klik tombol di bawah untuk membuat salinan template Google Sheet untuk soal ujian di Google Drive Anda. Anda bisa langsung mengisi soal di file salinan tersebut.

Buat Salinan Template Soal

Penting: Setelah disalin, jangan lupa atur setelan berbagi menjadi "Siapa saja yang memiliki link".

Langkah 2: Siapkan Spreadsheet Hasil & Skrip

Sekarang, buat salinan template untuk menampung hasil ujian siswa. Template ini sudah berisi skrip otomatis di dalamnya.

Buat Salinan Template Hasil

Setelah disalin, ikuti petunjuk ini untuk mengaktifkan skrip:

  1. Buka file salinan, lalu klik menu Ekstensi → Apps Script.
  2. Di halaman editor skrip, klik tombol biru Deploy → Deployment baru.
  3. Klik ikon gerigi (⚙️) di sebelah kiri "Pilih jenis", lalu pilih Aplikasi Web.
  4. Pada "Siapa yang memiliki akses", ubah menjadi Siapa saja.
  5. Klik Deploy, lalu klik Izinkan akses dan ikuti proses otorisasi dengan akun Google Anda.
  6. Setelah selesai, salin URL Aplikasi Web yang ditampilkan.

Langkah 3: Buat & Bagikan Link Ujian

Setelah kedua URL di atas terisi, klik tombol di bawah ini untuk mendapatkan link ujian final yang siap dibagikan kepada siswa Anda.

Simpan dan bagikan link ini kepada siswa Anda.
